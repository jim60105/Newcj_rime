# 自由大新倉頡RIME輸入方案

## 字典修改

- 移除
  - 網址
  - email
  - 電話
  - 表符
  - 常用語

## 安裝方法

將以下檔案拷貝到`%AppData%\Rime`目錄下，執行重新部署，即可在選單內找到該輸入法

### 自由大新

- newcj.schema.yaml
- newcj.dict.yaml

### 個人化修改

自由大新<=>注音互相反查(以 ` 啟用反查)\
啟用詞句輸入(以'斷字)\
部分標點改為半形輸出

- bopomofo.custom.yaml
- newcj.custom.yaml

以標點功能新增了我自己的常用語，若不需要請由`newcj.custom.yaml`中移除
