# 自由大新倉頡RIME輸入方案
## 安裝方法
自由大新 RIME輸入方案 : 
- newcj.schema.yaml
- newcj.dict.yaml

自由大新+傳統倉頡 RIME輸入方案 : 
- newcj+cj.schema.yaml
- newcj+cj.dict.yaml

自由大新、自由大新+傳統倉頡<=>注音互相反查(以 ` 開頭呼叫)： 
- bopomofo.custom.yaml
- newcj.custom.yaml
- newcj+cj.custom.yaml

將以上檔案COPY到`%USERPROFILE%\AppData\Roaming\Rime`目錄下，執行重新部署，即可在選單內找到該輸入法
