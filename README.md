# 自由大新倉頡RIME輸入方案

## 字典修改
- 移除所有網址
- 移除所有email和電話
- 新增 // -> https://

## 安裝方法
將以下檔案拷貝到`%AppData%\Rime`目錄下，執行重新部署，即可在選單內找到該輸入法

### 自由大新: 
- newcj.schema.yaml
- newcj.dict.yaml

### 自由大新+傳統倉頡: 
- newcj+cj.schema.yaml
- newcj+cj.dict.yaml

### 個人化修改
自由大新<=>注音互相反查(以 ` 開頭呼叫)<br>
啟用句式輸入<br>
一次空白斷字，再一次上屏<br>
部分標點改為半形輸出<br>
- bopomofo.custom.yaml
- newcj.custom.yaml
